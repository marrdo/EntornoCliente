Arrays

Los arrays son una estructura de datos muy simple en la que los elementos se encuentran contiguos en memoria. Esto hace que su acceso sea muy rápido y directo.

Los arrays se declaran vacíos o directamente con datos usando para ello los corchetes.

	let array1 = [];
	let array2 = [1,2,3, "Juan"];

A los arrays se accede por su posición y su longitud (la posición que ocupa el último elemento + 1) se encuentra en la propiedad length.

	array2[1] = 'Paco';                // [1, 'Paco', 3, "Juan"];
	console.log( array2[1] );          // 'Paco'
	console.table(array2);
	let longitudArray = array2.length; // 4

Javascript no se quejará ni mostrará un error si accedemos a una posición inválida del array (entendiéndose inválida como una posición no ocupada por datos). 

	array2[-1] = "Hola";
	array2[35] = "Caracola";
	alert( array2.length );   //36 --> posición del último elemento + 1

Trabajar con arrays

Recorridos

Dos métodos:

	let a = [1,2,3];

	for(let i=0; i<a.length; i++){
		console.log( a[i] ); 
	}

	for( let elemento of a){
		console.log( elemento );
	}

Añadir elementos

Se puede añadir un elemento por el final del array (push) o por el principio (unshift).

Se puede extraer un elemento por el final del array (pop) o por el principio (shift).

	let arr = [1,2,3];
	arr.push( 4 );           // [1,2,3,4]
	arr.unshift( 0 );        // [0,1,2,3,4]
	let últimoElemento = arr.pop();   // [0,1,2,3]  --> 4
	let primerElemento = arr.shift(); // [1,2,3]    --> 0
